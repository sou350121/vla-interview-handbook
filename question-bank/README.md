## 刷题建议
- **不要死记硬背**: 机器人面试更看重**工程直觉 (Engineering Intuition)**。
- **多讲 "为什么"**: 为什么选这个模型？为什么用这个 Loss？
- **结合项目**: 把问题往你做过的项目上引，展示你在真机上遇到的坑和填坑过程。
