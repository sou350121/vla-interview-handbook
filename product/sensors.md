# 触觉与感知 (Tactile & Sensors)

本章节汇总了赋予机器人"触觉"的关键传感器组件，涵盖了从工业级高精度传感器到开源低成本方案。

## Qianjue (千觉机器人)
> **Slogan**: "赋予机器人比人类更灵敏的触觉。"
> **官网**: [http://qianjue.jinjiayun.net.cn/](http://qianjue.jinjiayun.net.cn/)

### **XENSE-G1 (多模态触觉传感器系列)**
- **详情**: [产品页面](http://qianjue.jinjiayun.net.cn/product/367)
- **定位**: 通用型高精度触觉传感器平台。
- **核心参数**:
    - **分辨率**: **50,000 点/cm²** (信息密度是人类手指的 800 倍)。
    - **精度**: X/Y 轴 **0.03mm**, Z 轴 **0.06mm**。
    - **延迟**: 深度场处理时间仅 **10ms**。
- **多模态感知**:
    - **三维力**: 能同时感知法向力 (Pressure) 和切向力 (Shear/Friction)。
    - **纹理与材质**: 能识别物体表面的微小纹理 (如织物、金属拉丝)。
    - **滑移检测**: 毫秒级检测物体滑落趋势，实现动态握力调整。

### **G1-WS (Wedge Shape / 夹爪专用型)**
- **详情**: [产品页面](http://qianjue.jinjiayun.net.cn/product/386)
- **设计**: **楔形结构**。
    - 前端最薄处仅 **5mm**，专为狭窄空间作业设计 (如密集排布的线缆插拔)。
- **适配**: 完美适配 Robotiq 2F-85/140, Agile Robots 等主流夹爪。
- **应用**:
    - **盲插**: 在视觉被遮挡的情况下，靠触觉反馈完成 USB/网线插入。
    - **精密装配**: 轴承、齿轮等精密零部件的装配。

### **XENSE-Fingertip (指尖型)**
- **详情**: [产品页面](http://qianjue.jinjiayun.net.cn/product/368)
- **设计**: 仿生指尖曲面设计，模拟人类手指腹。
- **用途**: 适用于灵巧手 (Dexterous Hand) 指尖，增强抓取的稳定性和感知能力。

### **G1-OS (柔性层替换件)**
- **定义**: G1 系列传感器的**可更换柔性表皮**。
- **特点**:
    - **耗材化设计**: 表面磨损后可快速更换 (30秒内)，无需更换整个传感器。
    - **自标定**: 更换后支持自动校准，保证数据一致性。
    - **低成本维护**: 大幅降低了工业场景下的长期使用成本。

### **Xense_Sim (触觉仿真工具)**
- **平台**: 基于 NVIDIA Isaac Sim。
- **功能**:
    - **高保真仿真**: 精确模拟切向力 (摩擦力) 和法向力。
    - **Sim-to-Real**: 填补了触觉数据在仿真中的空白，加速策略训练。

## 9DTact
> **来源**: 学术成果 / 开源项目
> **参考**: [Project Page](https://github.com/Rwsl/9DTact)

- **特点**: **超紧凑 (Compact)**。
    - 尺寸仅为 **32.5mm x 25.5mm x 25.5mm** (约为 DTact 的 22%)。
- **实战表现 (Actual Situation)**:
    - **3D 重建精度**: 平均绝对误差 (MAE) 约 **0.046mm**，能清晰重建接触面几何细节。
    - **6D 力估计**: 基于神经网络 (训练集含 100k+ 数据对)，能同时解算 1D 法向力 + 2D 切向力 + 3D 扭矩。
    - **滑移检测**: 在抓取任务中，能通过切向力变化实时检测物体滑移 (Slip Detection)。
- **局限性**:
    - **光照敏感**: 作为视觉方案，尽管有遮光设计，强环境光仍可能干扰读数。
    - **标定依赖**: 需要对相机畸变和凝胶形变进行标定。
- **优势**: 适合安装在空间受限的灵巧手手指上，且开源成本低。

## Tac3D
> **来源**: 学术成果
> **参考**: [arXiv Paper](https://arxiv.org/abs/2308.08866)

- **原理**: **虚拟双目视觉 (Virtual Binocular)**。
    - 创新性地利用镜面反射，在单相机视野内构建双目视角，从而解决传统单目视触觉传感器的深度感知难题。
- **优势**: 结构简单，但能提供更准确的 3D 变形信息。

## 触觉传感器技术路线对比 (Technology Comparison)

本节基于业内主流技术路线 (ITPU/霍尔阵列、光学视触觉、传统六维力、压阻/电容) 进行深度对比分析。

| 特性 (Feature) | **光学视触觉** (如 GelSight, Daimon) | **ITPU / 霍尔阵列** (如 PaXini) | **压阻/电容阵列** (传统电子皮肤) | **传统六维力** (应变片式) |
| :--- | :--- | :--- | :--- | :--- |
| **核心原理** | **光-电转换** (Camera + Gel) | **磁-电转换** (6D Hall Array) | **接触电阻/电容变化** | **金属应变片形变** |
| **触点类型** | **连续曲面** (Continuous) | **多点面阵式** (Multi-point) | 多点阵列 | **单点式** (Single Point) |
| **结构形态** | 弹性体 + 刚性相机模组 | **柔性表面** (Flexible) | 薄膜 / 刚性 | 刚性 (Rigid) |
| **感知维度** | **3D 拓扑** + 剪切力 (间接) | **6维力 + 纹理/材质** (15维+) | 主测法向力 (Normal Force) | 仅 6维力 (Fx,Fy,Fz,Mx,My,Mz) |
| **空间分辨率** | **极高** (微米级, Micron-level) | **高** (0.1mm 级) | 中/低 | **无** (不具备空间分辨能力) |
| **响应速度** | **低** (30-120Hz, 受限于相机) | **高** (~1000Hz) | 高 | 高 (~1000Hz) |
| **迟滞 (Hysteresis)** | **高** (凝胶回弹慢) | **低** | 高 (压阻) / 低 (电容) | 极低 |
| **抗干扰能力** | **中** (受环境光/污渍影响) | **优** (软硬磁屏蔽优化) | 弱 (易受电磁干扰) | 优 |
| **耐久度 & 寿命** | **低** (凝胶易磨损/划伤, 需定期更换) | **高** (>1000万次, 磁耦合无接触磨损) | 中 (材料易疲劳/漂移) | **极高** (金属弹性体, 工业级寿命) |
| **集成度 & 尺寸** | **低** (需光路空间, 体积大/厚) | **高** (SoC内置, 体积小) | 低 (需外部采集卡) | 中 (通常较大) |
| **计算负载** | **极高** (需实时图像处理) | **低** (直接输出数字信号) | 中 | 低 |
| **典型成本** | **高** (相机+算力成本) | **低/中** (百元/千元级) | 高 (系统集成成本高) | **极高** (万元级) |
| **适用场景** | 精密检测, 实验室环境 | **灵巧手皮肤, 全身覆盖** | 简单接触检测 | 工业臂末端力控 |

### 总结
- **ITPU (霍尔阵列)**: 综合性能最强，在**尺寸、成本、响应速度**和**多维感知**之间取得了最佳平衡，是目前人形机器人皮肤和灵巧手大规模量产的首选方案。
- **光学视触觉**: 拥有无与伦比的**空间分辨率** (微米级)，适合对纹理细节要求极高的精密操作或缺陷检测，但体积和算力需求限制了其在全身皮肤上的应用。
- **传统六维力**: 精度和线性度最高，是工业机器人力控的**金标准**，但无法感知接触分布，且价格昂贵，不适合作为皮肤使用。
